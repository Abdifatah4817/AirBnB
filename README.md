# NairobiStay — AirBnB Clone

A premium short-term rental platform for Nairobi, built with React, TypeScript, and Tailwind CSS.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
